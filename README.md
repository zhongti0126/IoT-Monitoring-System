# Smart IoT Monitoring System

## 📌 Overview
This project is a simple IoT-based monitoring system that collects sensor data and displays it through a web interface

## 🚀 Features
- Simulated sensor data (temperature, humidity)
- Backend API using Flask
- Real-time data display

## 🛠 Tech Stack
- Python (Flask)
- JSON
- HTML

## 🎯 Goal
This project is designed to practice IoT system architecture and backend development.
